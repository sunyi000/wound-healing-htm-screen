## Installation instructions



### Batch analysis with nextflow

- go to https://jupyterhub.embl.de
- `cd ./code/nextflow`
- run the nextflow script
  - `rm -rf work; nextflow run woundHealing.nf; cat work/*/*/.command.log; rm -rf .next*; rm -rf work`
